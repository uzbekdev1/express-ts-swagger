# Express Swagger

Building REST API with Express, TypeScript, Swagger
 
## Build
 
1. Install dependencies.

   ```sh
   npm install
   ```

2. Build the production server.

   ```sh
   npm build
   ```

3. Run the server.
   ```sh
   npm start
   ```

## Test

```sh
npm test
```

## Docker

```sh
docker-compose build
docker-compose up
```
