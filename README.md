# Express TypeScript Swagger

Source code for the **Building REST API with Express, TypeScript, Swagger** 
 
## Build from source
 
1. Install dependencies.

   ```sh
   npm install
   ```

2. Build the production server.

   ```sh
   npm build
   ```

3. Run the server.
   ```sh
   npm start
   ```

## Build Docker image locally

```sh
docker-compose build
docker-compose up
```

## Run tests

```sh
npm test
```
